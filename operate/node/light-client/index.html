How to Run a Light Client

This guide outlines how to set up an Avail light client, covering both pre-compiled releases and source compilation.

Before you start

Ensure that you meet the system requirements. We recommend downloading the pre-compiled binary for speed and convenience.

Option 1: Run the Pre-Built Release​

All you need to do is run:

./avail-light -c config.yaml

Refer to the configuration reference for configuration options. See the "Using Avail" section for guides on embedding the light client.

Option 2: Build From Source​

To compile the client source code, run:

cd avail-light
cargo run

The client creates a default config.yaml file, initially unconnected to any network.

Once the build is complete, run:

./avail-light -c config.yaml

You can monitor the status of your node on the Avail Telemetry website.
