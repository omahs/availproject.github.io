Get Started Communicating with Avail

The API provides application developers the ability to query a node and interact with the Avail. Here you will find a list of examples in Javascript/Typescript, Rust and Go that show the basics of connecting to a local node, retrieving data from the node/chain and executing transactions on the chain.

Prerequisites​

For the following examples, you need to run a local node or update a configuration to match the running network. Detail instructions on how to run examples can be found in the Avail repository for Typescript for Go for Rust.

Examples​

Examples of Javascript/Typescript can be found in the Avail github repository and, there are also examples written in Go and Rust.

Some of the examples that show communication with Avail by submitting transactions and rpc calls:

  • Submitting a blob of data transaction example.
  • Dispatching data root transaction example.
  • Creating application key transaction example.
  • Creating a simple transfer example.
  • Query data proof example.
  • Query proof example.
  • Query app data example.
  • Listening to a new blocks example.
  • Getting chain and node information example.
